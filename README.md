# macbear_3d

Macbear 3d engine package project.

## Generate Shader

```
dart run build_runner build --delete-conflicting-outputs
```

## Generate UML

https://pub.dev/packages/dcdg
```
dart pub global run dcdg -s lib/src -o uml/src.puml
```

## Generate PNG/SVG

Extensions -> PlantUML
```
Export Workspace Diagrams -> png/svg
```

## Open Preview to the Side
https://open-vsx.org/extension/jebbs/plantuml
```
Top-Right-Icon -> Open Preview to the Side
```
